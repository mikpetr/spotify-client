# Spotify Client

For running app type
```
$ npm install
```
Then
```
$ npm install -g webpack
```
Then
```
$ npm install -g webpack-dev-server
```
Then
```
$ npm start
```

For Building
```
$ npm run build
```

For Testing
```
$ npm install -g karma-cli
```
```
$ npm test
```

For testing in development mode run
```
$ karma start
```
